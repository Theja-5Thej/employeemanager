Project Overview

This is an Employee Management Dashboard built using React.js that allows users to manage employee records efficiently. The application supports adding, editing, deleting, viewing, and printing employee details. It also includes search, filtering, status toggling, and profile image upload features. All data is persisted using browser localStorage, so no backend setup is required.

Tech Stack Used

React.js with TypeScript

Tailwind CSS

Material UI (for modals, switches, date picker)

Formik & Yup (form handling and validation)

Browser localStorage (data persistence)

Steps to Run the Project Locally

Clone the repository:

git clone <repository-url>


Navigate to the project directory:

cd employee-management


Install dependencies:

npm install


Start the development server:

npm run dev
